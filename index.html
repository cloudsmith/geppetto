geppetto by cloudsmith

Geppetto is an integrated toolset for developing Puppet modules and manifests.

Install

Cick on download and select the package for your platform. Unzip and run.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/cloudsmith/geppetto

get the source code on GitHub : cloudsmith/geppetto
